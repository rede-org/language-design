# Lists

## Variables

### Assignment

```
someList is {1, 2, 3},
```

```
someList is listA + listB,
```

### Declaration

```
someList: {int*}, {1, 2, 3};
```

```
someList: var, otherList;  `List type determined by initialization value.`
```

```
someList: listName, {1, 2, 3};
```

```
someList: {int*}, listA + listB;
```
