# Enums

The following code examples demonstrate how an enum type can be declared and used. See section \[TBD] for details.

## Type Declaration

```
enumName: [byte] A, B, C.
```

```
enumName: [int] A[100], B[101], C[102]
```

## Variable Assignment

```
e = enumName(B),
```

## Variable Declaration

```
e: enumName, A;
```
